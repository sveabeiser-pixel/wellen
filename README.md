# Lerninhalte: Arduino und Wellenlehre

Dieses Repository enthält interaktive Lernmaterialien zu zwei verschiedenen Themen: eine Einführung in den Arduino-Mikrocontroller und Übungen zur Wellenlehre. Die Inhalte sind für die direkte Nutzung im Webbrowser konzipiert und auf GitHub Pages gehostet.

## Arduino-Lernbaustein

Dies ist ein interaktiver Lernpfad, der schrittweise in die Grundlagen des Arduino-Mikrocontrollers einführt. Die Lektionen sind in zwei Hauptdateien aufgeteilt und basieren auf einem originalen PDF-Lernbaustein.

### Funktionen

- **Interaktive Aufgaben:** Jede Lektion enthält Aufgaben, deren Antworten direkt in Textfelder auf der Webseite eingegeben werden können.
- **Lokaler Fortschritt:** Deine Antworten und der Abschluss von Aufgaben werden automatisch im lokalen Speicher deines Webbrowsers gesichert. So kannst du jederzeit weitermachen, ohne Daten zu verlieren.
- **Keine Server-Anbindung:** Alle Interaktionen finden clientseitig statt. Es werden keine Daten an einen Server gesendet.
- **PDF-Integration:** Die ursprünglichen PDF-Seiten sind als Bilder eingebettet, um den Kontext zu wahren.

### Wie man es benutzt

1.  Öffne die Datei `arduino-start.html`, um mit den ersten Lektionen (Seiten 1–5) zu beginnen.
2.  Fahre anschließend mit `arduino-weiter1.html` fort, um die Lektionen der Seiten 6–11 zu bearbeiten.

## H5P-Inhalte: Wellenlehre

Dieses Projekt enthält auch eine Sammlung von interaktiven H5P-Inhalten zum Thema Wellen (Physik). Diese Übungen sind in sich geschlossen und können ebenfalls direkt im Browser ausgeführt werden.

### Enthaltene Dateien

- `Wellen.html`
- `wellen-book..html`
- `wellen-einfuehrung..html`
- `Wellen-book..h5p` (Quelldatei für den H5P-Editor)

### Wie man es benutzt

Öffne eine der `wellen-*.html`-Dateien in deinem Webbrowser, um die entsprechenden interaktiven Inhalte anzuzeigen.
